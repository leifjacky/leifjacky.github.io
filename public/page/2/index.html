最短编辑距离 Golang

问题 给定两个字符串 a 和 b，只允许以下三种操作： 插入一个字符； 删除一个字符； 替换一个字符。 求：把 a 转换成 b 的最小操作次数，也就是所谓的最小编辑距离。 举例： “xy” => “xz”，只需要把 y 替换成 z，因此，最小编辑距离为 1。 “xyz” => “xy”，只需要删除 z ，因此，最小编辑距离为 1。 思路 本题与最长公共子序列有相似性。插入和删除其实是逆操作，操作与 ret[i-1][j]与 ret[i][j-1]有关，替换则是与 ret[i-1][j-1]有关。 代码 package main import "fmt" const ( maxDistance = 0x3fffffff ) type Row []int func MinDistance(str1, str2 string) int { r1, r2 := []rune(str1), []rune(str2) return minDistance(r1, r2) } func minDistance(r1, r2 []rune) int { l1, l2 := len(r1), len(r2) a := genArray(l1, l2) for i := 1; i <= l1; i++ { for j := 1; j <= l2; j++ { if r1[i-1] == r2[j-1] { // 无需进行操作 a[i][j] = a[i-1][j-1] } else { a[i][j] = min(a[i-1][j-1], min(a[i-1][j], a[i][j-1])) + 1 // 从三种操作中选择最优 } } } return a[l1][l2] } func genArray(l1, l2 int) []Row { a := make([]Row, l1+1) for i := 0; i <= l1; i++ { a[i] = make(Row, l2+1) for j := 0; j <= l2; j++ { a[i][j] = maxDistance if i == 0 || j == 0 { a[i][j] = max(i, j) } } } return a } func max(a, b int) int { if a > b { return a } return b } func min(a, b int) int { if a < b { return a } return b } func main() { a := "spiderman" b := "superman" fmt....

ubuntu 16.04 搭建shadowsocks服务器

Shadowsocks 优势在于能够同时为 Win、Mac、iOS、Android 等设备同时提供服务，使用 sock5 进行代理，相较于 VPN 稳定高效。 安装 ss 环境，需要先安装pip： apt install -y python-gevent python-pip pip install --upgrade pip pip install setuptools pip install shadowsocks 写入配置文件/etc/shadowsocks.json： echo '{ "server":"0.0.0.0", "local_address": "127.0.0.1", "local_port":1080, "port_password": { "18080": "password" }, "timeout":300, "method":"rc4-md5", "fast_open": false }' > /etc/shadowsocks.json 启动服务 ssserver -c /etc/shadowsocks.json -d start 配置开机自动启动： echo '#!/bin/sh -e ssserver -c /etc/shadowsocks.json -d start exit 0' > /etc/rc.local 参考资料： 官网：https://shadowsocks.org/ github 主页：https://github.com/shadowsocks

在ubuntu16.04下通过squid进行http/https代理

1. 服务端设置 安装服务并编辑配置文件 apt install squid vi /etc/squid/squid.conf 在 squid.conf 首行添加 http_access allow all 找到http_port 3128，将 3128 改为自己想要的端口。 # Squid normally listens to port 3128 http_port 3128 重启 squid 服务 service squid restart 观察日志，看是否正常。 tail /var/log/squid/access.log 2. 客户端设置 在客户端使用填写hostname:port为代理服务器。 命令行下可以使用 export http_proxy=hostname:port export https_proxy=hostname:port Just enjoy ！！！ 参考资料： squid 的简单介绍:https://www.cnblogs.com/cherishry/p/5706736.html
